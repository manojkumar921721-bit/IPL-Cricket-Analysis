# 🏏 IPL Cricket Data Analysis

An end-to-end Data Analytics project built using **Python, SQL, Statistics, and Power BI** to analyze Indian Premier League (IPL) matches from 2008–2019.

The project covers the complete analytics workflow, including data cleaning, exploratory data analysis (EDA), statistical analysis, SQL-based insights, and interactive Power BI dashboards.


## 📖 Project Overview

The Indian Premier League (IPL) is one of the world's largest T20 cricket tournaments. This project analyzes historical IPL data to uncover insights into:

- Team Performance
- Player Performance
- Batting Analysis
- Bowling Analysis
- Venue Analysis
- Match Trends

The project demonstrates the complete lifecycle of a data analytics project, from raw data processing to business intelligence dashboards.

## 🎯 Project Objectives

- Clean and preprocess raw IPL datasets.
- Perform Exploratory Data Analysis (EDA).
- Analyze batting and bowling performance.
- Evaluate team performance across seasons.
- Discover venue-wise match trends.
- Perform SQL-based analytical queries.
- Design interactive Power BI dashboards.
- Generate business insights for decision-making.

## 📂 Dataset Information

The project uses historical IPL datasets containing:

- Match Details
- Ball-by-Ball Deliveries
- Player Information
- Team Information

The cleaned datasets were prepared using Python (Pandas) before importing into SQL and Power BI.

## 💡 Key Business Insights

- IPL has evolved into a high-scoring tournament with an average first-innings score of over 160 runs.
- A small group of players consistently dominate batting and bowling statistics.
- Certain venues host a significantly higher number of matches than others.
- Winning the toss alone does not guarantee winning the match.
- Team performance varies considerably across seasons.
